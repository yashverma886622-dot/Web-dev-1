## Responsive Portfolio Website (CSS Advanced)

This project is a one-page responsive portfolio website built as part of **Lab 3** for Web Development I.  
The website is created using **Flexbox**, **CSS Grid**, **Media Queries**, and **CSS Animations** to ensure a polished and responsive user experience across devices.

---

##  Features

### Flexbox  
Used for layout in Header and Hero sections to align elements properly.

### CSS Grid  
Used in Skills and Projects sections to create clean and structured card layouts.

### Responsive Design  
Website adapts to:
- Mobile (375px)
- Tablet (768px)
- Desktop (1024px+)

### Relative Units  
Used `rem`, `%`, `vh`, and `vw` for better scalability.

### Animations & Hover Effects  
- Keyframe animation for role text  
- Fade-in animations  
- Smooth hover effects on buttons & cards
